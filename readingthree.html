Reading One
How to Learn CSS

Five important skills:
  • > Memorize the basics. Know the terminology and language so you don’t waste a bunch of your time.
  • > Understand the types of selectors
  • > Organize your document. Keep it so it’s easily read and has a “flow” so thta others can also understand it.
  • > Use the box model to make your content organized for the viewer/ user
  • > Use transforms and animation sparingly
Questions
  • > Where does the name “Responsive Design” come from?
  • > If the designer specifies a specific font in CSS but a user doesn’t have it downloaded on their computer, will the font render or not?
